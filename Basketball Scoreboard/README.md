# 🏀 Basketball Scoreboard

A simple, clean, and responsive web-based Basketball Scoreboard designed to track scores for both **HOME** and **GUEST** teams. This project was built to practice foundational web development skills, semantic layout structuring, and dynamic DOM manipulation.

## 🌐 LIVE DEMO
Access the live [Basketball Scoreboard](https://6a289c29ea2ef0bef1e43a18--basketball-scoreboard-abdelaziz5.netlify.app/) directly on Netlify.

---

## ✨ Features
* **Dual Team Tracking:** Separate, clear columns for both **HOME** and **GUEST** scores.
* **Intuitive Controls:** Three distinct, easy-to-click buttons for each team to instantly add `+1`, `+2`, or `+3` points and a `reset` button to start counting for a new match.
* **Modern Typography:** Clean, scoreboard-style fonts imported dynamically from the web.
* **Responsive Design:** Styled using CSS Flexbox to ensure the board looks great on various screen sizes.

---

## 🛠️ Tech Stack & Concepts Used
* **HTML5:** Structured semantic markup for the scoreboard columns and control elements.
* **CSS3 (Flexbox):** Used for aligning the layout, centering the scoreboard, and creating a modern, sleek UI.
* **JavaScript (ES6):** Handles the score logic and dynamically updates the DOM upon user interaction.
* **Fonts:** External typography integration to give it an authentic sports arena feel.
